AI Hotel Booking System using Microservices, Docker, Kubernetes and AI Recommendation Engine

A cloud-native hotel booking platform developed using Spring Boot Microservices Architecture. The system includes room management, reservations, payments, reviews, AI-based room recommendations, API Gateway routing, Docker containerization, Kubernetes orchestration, secure authentication, and analytics dashboards.

Features
Microservices Architecture
REST API Communication
Spring Cloud API Gateway
AI Recommendation Engine
Secure Authentication with BCrypt
Payment Management
Review & Feedback System
Analytics Dashboard
Docker Containerization
Kubernetes Deployment
MySQL Database Integration
